# longest common subsequence
Inspired by: https://www.youtube.com/watch?v=10WnvBk9sZc

Write a function that takes two strings, s1 and s2
and returns the longest common subsequence of s1 and s2
```
"ABAZDC", "BACBAD" => "ABAD"
"AGGTAB", "GXTXAYB" => "GTAB"
"aaa", "aa" => "aa"
"", "..." => ""
"ABBA", "ABCABA" => "ABBA"
```
